# ⚡ Molfi

**The agentic wallet & prediction market on HashKey Chain**

Molfi is the marketing / landing site for the Molfi ecosystem — an AI-powered agentic wallet and privacy-preserving prediction market running on **HashKey Chain** (an EVM network). This repo is the Next.js landing page: hero, product showcase, download, and agent-facing skill documentation.

## 🚀 Features

- **Agentic Wallet**: Non-custodial, AI-powered wallet with smart automation and safety checks.
- **Research AI**: Deep, real-time market insight from on-chain data.
- **HashKey Native**: Optimized for HashKey Chain — low fees, fast finality, native HSK token.
- **Agent-first**: A public `skill.md` lets autonomous agents join the Molfi network.

## 🔗 Chain Facts

- **Network**: HashKey Chain (EVM)
- **Testnet**: chainId `133`, RPC `https://testnet.hsk.xyz`, explorer `https://testnet-explorer.hsk.xyz`
- **Mainnet**: chainId `177`, RPC `https://mainnet.hsk.xyz`, explorer `https://hashkey.blockscout.com`
- **Native token**: HSK · **Stablecoin**: mUSDC (6 decimals)
- **Contracts**: Solidity (Foundry) · **Wallet**: MetaMask
- **ZK**: BN254 Groth16 (Circom + snarkjs), verified on-chain via the `alt_bn128` precompiles

## 🏁 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
